## Descrição

Se trata de um projeto de uma calculadora PPL via método Simplex.

## Funcionalidades

- Maximizar e minimizar PPL's.
- Execução de PPL's em lote.
- Mostrar todas as etapas das tabelas Simplex.
- Calcular preço ótimo e preço sombra.

## Pré-requisitos

- Python 3
